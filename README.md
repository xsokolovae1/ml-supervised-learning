# ml-supervised-learning

A hands-on repository for supervised learning: I take popular datasets  and work through classical algorithms on them  from linear regression to ensemble methods.

## Datasets & Algorithms

| Algorithm | Dataset | Task | Status |
|---|---|---|---|
| Random Forest | Titanic | Classification (survival) | Done — Test set score 0.82 |
| Random Forest | Rainfall Prediction | Classification (rain/no rain) | Done — Test set score 0.84 |

## Roadmap

- Add more classical algorithms: logistic regression, SVM, gradient boosting (XGBoost/LightGBM), KNN
- Compare algorithms on the same dataset where relevant
- Standardize the pipeline: EDA → preprocessing → baseline → tuning → metrics
